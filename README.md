## Project

**MijnMotorParkeren.nl** is a free, map-based app for motorcyclists to find up-to-date parking rules, dedicated spots, and official sources across the Netherlands. Our goal: **make it easy for everyone to know how to park their motorcycle, anywhere in the country.**

Because the project is open-source and file-based, anyone can help keep the data accurate and transparent. Ready to see it in action? Head over to [mijnmotorparkeren.nl](https://mijnmotorparkeren.nl) and find out.

## Contributing

Anyone can help keep our data accurate, and you do not even need to clone or run the project locally to do it\!

The easiest way to contribute is to [open a GitHub issue](https://github.com/hawkinslabdev/mijnmotorparkeren/issues) to report an error, suggest an update, or submit missing information. If you prefer, you can submit a Pull Request with your changes directly, just be sure to include official sources and use ISO 8601 date formats. 

If you're spinning up a local environment, you can run `npm run validate:data` to test changes locally. All submissions will be reviewed before they are merged.

## Help

Need help using the site, have ideas, or found something incorrect? Your feedback makes MijnMotorParkeren.nl better for everyone.

To report bugs or request features, please [open an issue on GitHub](https://github.com/hawkinslabdev/mijnmotorparkeren/issues). If you have the fix, feel free to open a Pull Request so the changes can be merged quickly\! For general questions, feedback, or just to chat, you can start a discussion on the GitHub repository.
