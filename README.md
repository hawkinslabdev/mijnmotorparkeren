## Project

**MijnMotorParkeren.nl** is a free, map-based web application for motorcyclists to find up-to-date parking rules, dedicated spots, and their official sources across the Benelux. My goal? To **make it easy for everyone to know how to park their motorcycle, anywhere in the country.**

Because the project is open-source and file-based, anyone can help keep the data accurate and transparent! Even if you have no experience with changing the `JSON` files, you can submit an issue (see section below) without any technical knowledge required. Ready to see it in action? Head over to [mijnmotorparkeren.nl](https://mijnmotorparkeren.nl) and find out.

## Contributing

Anyone can help keep the data fresh! The easiest way is to [open a GitHub issue](https://github.com/hawkinslabdev/mijnmotorparkeren/issues) to report anything, whether that's an update on a specific area or something that looks off. 

Rather jump straight into the code? Submit a Pull Request with your changes, just make sure to include official sources and use ISO 8601 date formats. You can run `npm run validate:data` locally to test your changes before submitting. All submissions are reviewed before they get merged.

Got questions, ideas, or spotted something incorrect? Your feedback makes MijnMotorParkeren.nl better for everyone. Start a discussion on the GitHub repository and let's figure it out together.
